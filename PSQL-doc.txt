Основные команды в postgres: 
1) Удаление всей таблицы: DROP TABLE <table_name>
2) Изменение кодировки в cli psql: psql \! chcp 1251
3) Добавление колонки в таблицу: ALTER TABLE table_name ADD COLUMN column_name data_type
4) Удаление колонки в таблице: ALTER TABLE table_name DROP COLUMN column_name data_type
